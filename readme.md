# FAQ
## 如何减小 Visual Studio 工程文件大小
将 Visual Studio 工程文件中的下列文件夹删掉可以大幅度减少工程压缩包体积。
1. ```.vs``` Visual Studio 临时信息储存目录
2. ```Debug``` 默认调试编译目录
3. ```Release``` 默认发布编译目录
