## 第十章练习题 10
设一个整数数组 ```a[0..n-1]``` 中存在互不相同的 ```n``` 个整数，且每个元素的值均在 ```1~n``` 之间。设计一个算法在 ```O(n)``` 时间内将 ```a``` 中的元素递增排序，将排序结果放在另一个同样大小的数组 ```b``` 中。
